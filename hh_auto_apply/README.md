# HH Auto Apply

Автоматизированный инструмент для отклика на вакансии на hh.ru (HeadHunter)

## Обзор

HH Auto Apply - это Python-приложение, которое автоматизирует процесс отклика на вакансии на hh.ru. Оно предоставляет функции поиска вакансий по критериям, автоматического отклика на найденные позиции и мониторинга статуса откликов.

## Возможности

### Основной функционал
- **Поиск вакансий**: Поиск вакансий по ключевым словам, зарплате, местоположению, опыту, типу занятости и графику работы
- **Автоотклик**: Автоматический отклик на найденные вакансии с ограничением скорости (20 откликов в минуту)
- **Ручной отклик**: Отклик на конкретные вакансии из списка
- **Отклик по рекомендациям**: Отклик на вакансии из рекомендаций hh.ru
- **Шаблоны сопроводительных писем**: Использование шаблонов для персонализированных писем с поддержкой A/B тестирования

### Управляющие функции
- **Управление резюме**: Выбор и обновление резюме для откликов
- **Мониторинг откликов**: Отслеживание статуса откликов и получение уведомлений
- **Управление данными**: Хранение отвеченных вакансий, поисковых запросов и экспорт отчетов
- **Множественные профили**: Поддержка различных конфигурационных профилей

### Технические функции
- **Headless браузер**: Автоматизация Chrome браузера с мерами против обнаружения
- **Ограничение скорости**: Соблюдение ограничений hh.ru для избежания блокировки
- **Управление сессиями**: Сохранение и восстановление сессий браузера
- **Обработка ошибок**: Механизмы повторных попыток и восстановления после ошибок
- **Логирование**: Полное логирование для отладки и мониторинга

## Установка

### Требования
- Python 3.8 или выше
- Браузер Google Chrome
- ChromeDriver (управляется автоматически)

### Настройка

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/yourusername/hh-auto-apply.git
   cd hh-auto-apply
   ```

2. Создайте виртуальное окружение:
   ```bash
   python -m venv venv
   source venv/bin/activate  # В Windows: venv\Scripts\activate
   ```

3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

4. Настройте приложение:
   ```bash
   cp config/default.json config/my_config.json
   # Отредактируйте config/my_config.json с вашими учетными данными и настройками
   ```

## Использование

### Базовое использование
```bash
python main.py --mode auto --config config/my_config.json
```

### Параметры командной строки
```
--mode {auto,manual,recommendations}
    Режим работы приложения (по умолчанию: auto)
    
--config CONFIG
    Путь к файлу конфигурации (по умолчанию: config/default.json)
    
--keywords KEYWORDS
    Ключевые слова для поиска
    
--area AREA
    Код региона/области
    
--salary SALARY
    Минимальная зарплата
    
--experience {noExperience,between1And3,between3And6,moreThan6}
    Уровень опыта
    
--vacancy-file VACANCY_FILE
    Файл с ID вакансий (для ручного режима)
    
--limit LIMIT
    Ограничение скорости (откликов в минуту, по умолчанию: 20)
    
--resume-id RESUME_ID
    Конкретный ID резюме для использования
    
--monitor
    Мониторинг статуса откликов
    
--export EXPORT
    Экспорт результатов в файл
    
--dry-run
    Выполнить пробный запуск без реальных откликов
    
--verbose, -v
    Включить подробный вывод
```

### Конфигурация

Создайте файл конфигурации на основе `config/default.json`:

```json
{
  "credentials": {
    "username": "your_email@example.com",
    "password": "your_password"
  },
  "application": {
    "headless": true,
    "rate_limit": 20,
    "delay_range": [1, 3],
    "dry_run": false
  },
  "search_filters": {
    "text": "Python developer",
    "area": "1",
    "salary": 100000,
    "experience": "between3And6",
    "employment": ["full"],
    "schedule": ["fullDay"]
  },
  "resume": {
    "default_id": "your_resume_id",
    "auto_update": true
  }
}
```

### Ручной режим

Создайте файл с URL вакансий или ID (по одному на строку):
```
https://hh.ru/vacancy/12345678
87654321
```

Затем запустите:
```bash
python main.py --mode manual --vacancy-file data/manual_vacancies.txt --config config/my_config.json
```

## Модули

### Основные модули
- `application.py`: Главный класс приложения
- `config_manager.py`: Управление конфигурацией
- `session_manager.py`: Управление сессиями браузера

### Функциональные модули
- `auth_module.py`: Аутентификация на hh.ru
- `search_module.py`: Функциональность поиска вакансий
- `apply_module.py`: Функциональность отклика на вакансии
- `resume_module.py`: Управление резюме
- `monitor_module.py`: Мониторинг статуса откликов

### Утилиты
- `browser_utils.py`: Вспомогательные функции браузера
- `data_utils.py`: Работа с данными и операции с базой данных
- `logger.py`: Настройка логирования

- `cli_interface.py`: Интерфейс командной строки

- Требуются действительные учетные данные аккаунта hh.ru
